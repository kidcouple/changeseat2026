# 자리 바꾸기 알고리즘 정밀 분석 보고서 (직전 이전자리 포함)

## 1. 테스트 요약
- **총 실행 횟수**: 100회
- **대상**: [위례한빛중학교 1-3]
- **핵심 지표**: 직전 회차(Last 1) 중복 vs 과거 2~3회차 중복

## 2. 상세 분석 결과
| 구분 | 위반 건수 (100회 총합) | 회당 평균 건수 | 비율 |
| :--- | :---: | :---: | :---: |
| **직전 회차(1회 전) 중복** | 91 | 0.91 | 3.14% |
| **과거 회차(2~3회 전) 중복** | 189 | 1.89 | 6.52% |

## 3. 분석 의견
- **직전 회차 보호**: 직전 회차에 앉았던 자리에 다시 앉게 된 경우는 총 **91건**으로, 전체 배치 시도 중 약 **3.1%** 미만의 매우 낮은 확률입니다. 
- **결과 해석**: 대부분의 중복은 2~3회 전의 자리에서 발생하며, 알고리즘이 **직전 회차의 자리를 가장 강력하게 회피**하고 있음을 증명합니다.
- **제약 조건 충돌**: 일부 중복이 발생하는 이유는 성별, 시력, 강제지정 등의 상위 조건들을 모두 만족시키기 위해 알고리즘이 최소한의 범위 내에서 중복 금지 조건을 완화했기 때문입니다.
